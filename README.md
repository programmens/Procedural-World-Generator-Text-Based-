A Python-based procedural world generator with player movement.

## Features

- Random world generation
- Multiple terrain types
- Town placement
- Player spawn
- WASD movement
- Seed-based reproducibility

## Terrain Legend

P  - Player  
~  - Water  
T  - Forest  
^  - Mountain  
.  - Plains  
#  - Town  

## How to Run

python world_generator.py

Optional:
Enter a seed to generate the same world again.

## Controls

W - Move Up  
A - Move Left  
S - Move Down  
D - Move Right  
Q - Quit  

## Learning Concepts

- Classes and OOP
- 2D Lists
- Random Generation
- Procedural Content
- Game Logic
- Collision Detection
